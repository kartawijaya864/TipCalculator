# Tip Calculator Android App

## Deskripsi

Tip Calculator adalah aplikasi Android sederhana yang dibuat menggunakan Jetpack Compose. Aplikasi ini digunakan untuk menghitung jumlah tip berdasarkan nilai tagihan dan persentase tip yang dimasukkan oleh pengguna.

## Fitur

* Menghitung tip secara otomatis
* Input jumlah tagihan (Bill Amount)
* Input persentase tip (Tip Percentage)
* Opsi pembulatan tip (Round Up Tip)
* Antarmuka menggunakan Jetpack Compose

## Teknologi yang Digunakan

* Kotlin
* Jetpack Compose
* Material 3
* Android Studio

## Cara Menjalankan

1. Clone repository ini.
2. Buka project menggunakan Android Studio.
3. Sync Gradle.
4. Jalankan aplikasi pada emulator atau perangkat Android.
